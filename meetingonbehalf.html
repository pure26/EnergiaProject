Set up a Skype for Business meeting on behalf of someone else

COMPUTER OPERATIONS DEPT / MOBILITY SUPPORT GROUP

Get set up as a delegate

First, the steps below must be done by the person who is giving delegate access.

1. In Outlook, click File > Account Settings > Delegate Access > Add
2. Find and add the name of the person who is going to be the delegate.
3. Click the Calendar menu, and select Editor rights.



After that kill both apps Outlook and Skype and re-open them again

Check the skype client if the delegate is showing.


From the delegate side, it should also appear under list “people I manage calls for” as below screenshot.

If the delegator is not showing, reboot the computer and redo the same steps from the delegator machine after removing the new delegate access from both Skype and Outlook.

After that open registry key and remove the following item.

Computer\HKEY_CURRENT_USER\Software\Microsoft\Office\16.0\Lync\khalid.alotaibi.30@aramco.com

Then delete all files indie the “Tracing” folder following this path:

C:\Users\alotka0g\AppData\Local\Microsoft\Office\16.0\Lync

Finally, redo the same steps again for adding a new delegate from Outlook as explained above.
